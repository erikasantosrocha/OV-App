🔗 Google Sheets + URL
⚡ La URL de Apps Script se usa también para verificar la clave de acceso.
📋 Guía de Configuración
1
Abre Google Sheets, crea hoja llamada OVApp.
2
Extensiones → Apps Script. Borra el código y pega el de abajo (incluye la clave).
// ============================================= // OV APP — APPS SCRIPT BACKEND // PASO 1: Cambia la clave en la línea de abajo // ============================================= var APP_PASSWORD = "tu_clave_aqui"; // CORS helper — required for browser requests function corsResponse(data) { return ContentService.createTextOutput(JSON.stringify(data)) .setMimeType(ContentService.MimeType.JSON); } // GET — lee datos (login + carga) function doGet(e) { var pass = e.parameter.password; if (pass !== APP_PASSWORD) { return corsResponse({ok: false, error: "unauthorized"}); } // Special action: just verify password if (e.parameter.action === "ping") { return corsResponse({ok: true}); } var ss = SpreadsheetApp.getActiveSpreadsheet(); var sheetName = e.parameter.sheet; var sheet = ss.getSheetByName(sheetName); if (!sheet) return corsResponse({rows: []}); return corsResponse({rows: sheet.getDataRange().getValues()}); } // POST — escribe datos function doPost(e) { var data = JSON.parse(e.postData.contents); if (data.password !== APP_PASSWORD) { return corsResponse({ok: false, error: "unauthorized"}); } var ss = SpreadsheetApp.getActiveSpreadsheet(); var sheet = ss.getSheetByName(data.sheet) || ss.insertSheet(data.sheet); if (data.action === "replace_all") { sheet.clearContents(); sheet.appendRow(data.headers); data.rows.forEach(function(r) { sheet.appendRow(r); }); } return corsResponse({ok: true}); }
3
En la línea var APP_PASSWORD = "ovladrillera2024" cambia ovladrillera2024 por tu clave real.
4
Implementar → Nueva implementación. Tipo: Aplicación web. Acceso: Cualquier usuario. Implementar.
5
Copia la URL generada, pégala en el campo de arriba y también en la pantalla de login de la app.
⚠️ Zona Peligrosa

Borra todos los datos locales. Irreversible.
